# Furnando-hair-assistant-
A aplicação analisa fotos ou vídeo da pessoa de frente e de perfil para identificar o formato do rosto e da cabeça. Com base nessa análise e num pequeno questionário (ex.: cabelo curto, comprido ou com franja), recomenda cortes adequados e pode gerar imagens realistas da pessoa com os estilos sugeridos usando IA.
